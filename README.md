# Acordeon en linea

Se puede tocar acordeon de sol, fa, y mi con esta aplicacion.

![](./screenshot.png)

## Como correr el programa

```bash
npm install
npm run dev
```

Ir a [localhost:8080](http://localhost:8080).

## Para generar la logica de produccion (para subir la logica a un servidor):

```bash
npm run build
```
